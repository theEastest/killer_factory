# 初步实现状态

本轮目标是提供可构建、可加载、可继续迭代的玩法垂直切片，并非完成任务书全部验收项。

## 已实现

- 初始牌组：5 张基础攻击构件、5 张基础防御构件、1 张手工装配。
- 初始遗物“简易机械臂”，战斗开始时初始化产线状态。
- 三组回路牌：P（生产）、C（消费）、D（分解）。
- 回路从弃牌堆返回最近的不同名搭档；手牌满时立即中断。
- 废料是实际状态牌，会占用手牌位置；消费者和计划报废可生成废料。
- 回收工单会分解手牌中最左侧废料，获得能量并抽牌。
- 机械原料与化合原料以战斗内仓储计数实现。
- 手工装配简化融锻：自动选择最左侧两张合法构件，升级主体并消耗材料；普通主体只能融锻一次。
- 战斗 UI 上方的中文色块机械产线面板。
- 卡牌、角色、遗物均有 SVG 色块占位美术和中文文字。

## 下一阶段

- 手工装配的两步手动选择，以及真正合并费用、效果与升级状态。
- 机械槽位的卡牌拖放、取回、单击启动和随机敌人机械打出。
- 加工台三槽位与全部加工词条。
- 原料改为仓储牌堆/实体对象，并支持分解、投料和修复。
- 精密、脆弱、病毒、一次性、损坏品等完整生命周期。
- 产线暂停、继续、加速以及多人确定性测试。

## 当前设计差异

- 反应模具升级暂时改为额外生产 1 份化合原料，因为原料尚未进入实体仓储牌堆。
- 回收工单升级暂未提供原料选择。
- 手工装配使用确定性的“最左侧两张”规则，等待卡牌选择 UI 接入。
- 简易机械臂本轮只负责架设与状态展示，尚不能装填和自动打出。
